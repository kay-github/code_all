# 大杂烩工具站

一个移动端优先的轻量 H5 工具站。当前已落地独立子功能：错别字校对。

## 产品方案

### 工具定位

- 面向手机端快速使用的极简工具。
- 用户通过独立链接直接进入具体功能页，无需从首页跳转。
- 页面只保留输入、校对、结果、复制四个核心动作。

### 错别字校对规则

- 仅修正错别字、多字、漏字和明显错误标点。
- 不调整原文句式、语序、用词和表达逻辑。
- 不做润色、扩写、缩写或风格调整。
- 无错时原样返回。
- 结果区域只展示修正后的完整文本。

### 移动端交互

- 大高度输入框，适合粘贴长文本。
- 主按钮固定为单一核心动作：开始校对。
- 校对中禁用按钮并显示加载状态。
- 空内容提交给出轻提示。
- 结果支持一键复制，并反馈复制成功。

## 技术方案

- 选型：原生 HTML + CSS + JavaScript。
- 部署：纯静态文件，无构建步骤，无第三方依赖。
- 路由：
  - `/index.html`：工具站入口。
  - `/tools/typo-proofreader/index.html`：错别字校对独立页面。
- API：前端预留大模型接口配置位；正式上线建议由后端保存 API Key，H5 只请求自己的后端接口。

## 开发规则

- 保持代码精简，基础功能不用重框架和复杂状态管理。
- 移动端优先，按钮高度不低于 48px，避免横向滚动。
- 页面功能只围绕核心任务，不增加弹窗、广告、历史记录、登录等非必要能力。
- 所有 API 配置集中在页面脚本顶部，便于后续替换。
- 不在仓库中提交真实 API Key、令牌或私密配置。
- 校对提示词必须保持强约束，禁止模型进行润色或解释。

## 本地运行

直接用浏览器打开：

- `index.html`
- `tools/typo-proofreader/index.html`

也可以放到任意静态服务器中访问。
